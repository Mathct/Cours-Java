## Semaine 5 – Authentification simple + consolidation

Objectif global : sécuriser ton backend avec Spring Security + JWT et finaliser une version “prête à livrer”.

---

## Compétences visées

- Comprendre le pipeline Spring Security.
- Authentifier un utilisateur et générer un JWT.
- Protéger les endpoints métiers avec `Bearer` token.
- Gérer rôles et permissions de base.
- Tester les scénarios sécurité essentiels.

---

## Plan détaillé

- **Jour 1** : bases Spring Security.
- **Jour 2** : modèle utilisateur + endpoint login.
- **Jour 3** : service JWT (génération, validation).
- **Jour 4** : sécuriser `/tasks` avec token.
- **Jour 5** : rôles (`USER`/`ADMIN`) et autorisations.
- **Jour 6** : tests sécurité + hardening.
- **Jour 7** : finalisation, packaging et checklist de livraison.

---

## Livrable de fin de formation

Une API Java/Spring Boot complète avec :

- persistance SQL,
- validation/erreurs propres,
- sécurité JWT,
- tests clés,
- build exécutable en JAR.
